Ooops! There has been a Fatal Error

This page could not be loaded ▼

What happened?
The page has caused a fatal error in Coward browser

What can you do?

If you are a developer:

  • Download Coward browser source code from here
  • Open Coward project with your IDE of choice (e.g. PyCharm)
  • Activate Debug Info in settings._defaultSettings, by setting to 'true' the enableDebug option inside App Section
  • Run Coward
  • Check in output console which page is producing the error (most likely is an AD or similar page)
  • When found, include the problemmatic page domain in settings, within urlBlackList option in AdBlocker section
  • Try again. The issue should be fixed (there can be more, nevertheless)

If you are not a developer or don't want to mess around this code, open an issue here
